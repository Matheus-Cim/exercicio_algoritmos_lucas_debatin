## 📌 Descrição
Programa em C++ que permite cadastrar até 5 alunos, calcular a média das três notas e exibir a situação (Aprovado/Reprovado), além de identificar o aluno com maior média.

## ⚙️ Funcionalidades
- Cadastro de alunos
- Cálculo de média
- Exibição de tabela
- Identificação do melhor aluno
